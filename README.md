Эта простенькая программа позволяет найти нужный general.bat для zapret-discord-youtube. Сейчас обьясню как её использовать:

1. Открываем программу
2. Нажимаем кнопку "Начать проверку"
3. Указываем путь до папки zapret
4. Просто нажимаем на кнопку "Да" в окне открытия от имени администратора до тех пор, пока они не перестанут появляться
5. Возвращаемся к основному окну программы и видим результат
6. Закрываем программу

Вот и всё!
